# Overview
The vSphere Automation SDKs bundle client libraries, documentation and sample
code for accessing the below services:

- NSX-T APIs*

## IMPORTANT NOTICE
Starting with vSphere 9.0.0.0 release version, the vSphere Automation Java SDK and vSphere Automation Python SDK repo only contains NSX on-prem libraries.

Please use [VCF SDK Java](https://github.com/vmware/vcf-sdk-java) or [VCF SDK Python](https://github.com/vmware/vcf-sdk-python)


# Getting the SDKs
The vSphere Automation SDK’s are available from VMware’s GitHub source
repositories. Simply choose the SDK for your preferred language, clone the repo
and follow the instructions in the README.md in the root folder of the repo to
be up and running in minutes!

For more information refer to the vSphere Automation SDK Overview.
